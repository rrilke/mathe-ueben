# Math Practice - Calculus Study Program

A browser-based educational program designed to help students practice basic math calculations.

## Overview

This interactive web application helps students practice:
- **Addition** - Practice adding numbers from 0 to 10
- **Subtraction** - Practice subtracting numbers from 0 to 10

### Features
- Choose between addition (+) or subtraction (−) operations
- 10 randomly generated questions per quiz
- Numbers range from 0 to 10
- Real-time score tracking
- Immediate feedback on answers
- Final results summary with performance message

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Running the Program

1. Open `index.html` in your web browser by:
   - Double-clicking the file, or
   - Right-clicking and selecting "Open with" → your browser, or
   - Dragging the file into your browser window

2. Select an operation (+ or −)
3. Click "Start Quiz"
4. Answer the 10 questions
5. View your results!

## How It Works

1. **Settings Page**: Choose between addition or subtraction
2. **Quiz Page**: Answer 10 randomly generated questions
3. **Results Page**: See your final score and performance message
